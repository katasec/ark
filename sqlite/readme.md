# Notes on SQLite

Use the `sqlite3` command line tool for interacting with the DB

# Start cmdline

```
sqlite3
```

# Opening a DB

```
.open ark.db
```

# Open in oneline

```
sqlite3 ~/.ark/db/ark.db
```

# List Tables

```
.tables
```

# Describe Table

```
.schema azurecloudspace
```

# Quit command line

```
.quit
```
